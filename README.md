# ISCI Security Culture Dataset

This repository contains supporting data for the research paper:

**"Beyond Security Awareness: A Mixed-Method Study of Behavioral and Organizational Factors Influencing Security Culture"**

---

## Overview

This dataset supports a mixed-method study conducted across organizations in Karachi, Pakistan.

A total of **100 participants** contributed:

- **20 participants** → Semi-structured interviews (qualitative)
- **80 participants** → Structured questionnaires (quantitative)

Participants were categorized into:

- Managers and Supervisors  
- Technical Staff (IT/Security)  
- Non-Technical Employees

## Key Research Themes

The analysis identified the following major themes:

1. Leadership Influence  
2. Security Behavior and Compliance  
3. Workload and Productivity  
4. Training and Awareness Effectiveness  
5. Technical Controls vs Human Behavior
